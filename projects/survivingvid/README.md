# SurvivingVid - AI 영상 생성 플랫폼

## 🎬 개요

AutoVid의 모든 기능을 완벽하게 복제한 AI 영상 생성 플랫폼입니다.

- **AI**: Gemini + Vertex AI 이미지 생성
- **UI**: Tabler 테마 + Toast UI Editor
- **배포**: GitHub + Vercel

## 🚀 기능

### 12개 핵심 메뉴
1. **로그인** - Google OAuth / Firebase
2. **다운로드** - YouTube 비디오 다운로드
3. **수동영상생성** - 타임라인 기반 영상 편집
4. **자동영상생성** - AI 기반 완전 자동 생성
5. **베스트시리즈** - 시리즈 영상 자동 생성
6. **Template** - 8개 기본 템플릿 + 에디터
7. **유튜브 탐색** - 내장 YouTube 브라우저
8. **Fonts Download** - Google Fonts 통합
9. **BGM** - Pixabay 음악 검색
10. **프로필 설정** - 개인 설정 관리
11. **Shop** - 크레딧 충전 및 구독
12. **Settings** - 앱 설정

## 🛠️ 기술 스택

- **프론트엔드**: Next.js 15 + TypeScript + Tabler
- **에디터**: Toast UI Editor
- **AI**: Gemini + Vertex AI Imagen
- **인증**: Firebase Auth
- **배포**: Vercel
- **버전 관리**: GitHub

## 📋 설치 및 실행

```bash
# 개발 환경 실행
npm run dev

# 빌드
npm run build

# 프로덕션 실행
```
